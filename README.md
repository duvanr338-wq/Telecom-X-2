# 📊 Análisis Exploratorio de Datos (EDA)

## 📌 Descripción
Este proyecto presenta un análisis exploratorio de datos (EDA) con el objetivo de comprender la estructura del dataset, identificar patrones, detectar valores atípicos y explorar relaciones entre variables.

El análisis permite obtener una visión clara de los datos antes de aplicar modelos estadísticos o de machine learning.

---

## 🎯 Objetivos

- Comprender la estructura general del conjunto de datos.
- Identificar patrones y tendencias relevantes.
- Detectar valores faltantes y outliers.
- Analizar la relación entre diferentes variables.
- Generar visualizaciones que faciliten la interpretación de los datos.

---

## 📂 Dataset

El dataset utilizado contiene múltiples variables que representan diferentes características del fenómeno analizado. Dependiendo del caso, estas variables pueden ser:

- Variables numéricas
- Variables categóricas
- Variables temporales
- Indicadores o métricas específicas

Durante la fase inicial se realizó:

- Limpieza de datos
- Manejo de valores faltantes
- Conversión de tipos de datos
- Transformación de variables cuando fue necesario
